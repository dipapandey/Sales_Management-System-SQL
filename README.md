# Sales Management System (SQL)

## Description
A basic Sales Management System using MySQL for storing and analyzing sales data.

## Features
- Create Database
- Create Table
- Insert Data
- Retrieve Data
- Sales Analysis using SQL

## SQL Concepts Used
- CREATE DATABASE
- CREATE TABLE
- ALTER TABLE
- INSERT
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- LIMIT
- SUM
- AVG
- COUNT

## Tools Used
- MySQL
- MySQL Workbench
